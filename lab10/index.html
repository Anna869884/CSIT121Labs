What is XML?

XML stands for eXtensible Markup Language. XML is a markup language much like HTML. The XML language has no predefined tags, it lets you define your own tags.

How is XML used?

  • XML was designed to store data and transport data
  • XML was designed to be self-descriptive
  • XML was designed to be both human- and machine-readable
  • XML separates information from presentation
  • XML simplifies data sharing, data transform, platform changes and data availability
  • XML is a W3C recommendation

What is JavaScript?

JavaScript is an object-based, client-side scripting language interpreted by a web browser. The JavaScript code is embedded in the HTML file and the web browser interprets it and displays the results as needed.
